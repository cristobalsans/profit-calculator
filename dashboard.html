Conectar Shopify

Crea una app personalizada en tu tienda para obtener el token permanente.

Cómo obtener el token (3 minutos):
  1. Shopify Admin → Configuración → Apps y canales de venta
  2. Click en Desarrollar apps → Crear una app
  3. Nombre: Profit Calculator → tab Configuración de la API Admin
  4. Activa read_orders y read_products → Guardar
  5. Click Instalar app → tab Credenciales de API
  6. Copia el Token de acceso de la API de Admin

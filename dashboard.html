Conectar Shopify

Crea una app personalizada en tu Shopify Admin para obtener un token permanente.

Cómo obtener el token (3 minutos):
  1. Entra a tu Shopify Admin → Configuración
  2. Click en "Apps y canales de venta"
  3. Click en "Desarrollar apps" (arriba a la derecha)
  4. Click en "Crear una app" → nombre: Profit Calculator
  5. Tab "Configuración de la API Admin" → activa read_orders y read_products → Guardar
  6. Click "Instalar app" → tab "Credenciales de API"
  7. Copia el "Token de acceso de la API de Admin"
→ Abrir Shopify Admin ↗
